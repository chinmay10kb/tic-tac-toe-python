# tic-tac-toe-python
A Tic-Tac-Toe game developed using python which can be played in terminal.

# Features:
-Two player mode
-Player 1 chooses a symbol 'x' or 'o', then automatically player 2 will get other symbol
-Input validation
-Detects:
 -Row wins
 -Column wins
 -Diagonal wins
 -Draws
-Uses simple python lists for the board:
l1 for row1
l2 for row2
l3 for row3

# Files:
-"tic_tac_toe.py"  -> Main python game file

# How to run:
-Install python 3
-Download or copy this repository:
```git clone https://github.com/chinmay10kb/tic-tac-toe-python.git```
-Open terminal inside project folder
-Run: python tic_tac_toe.py

# Game Instructions
- Player 1 chooses `'x'` or `'o'`
- Enter a **row number (1–3)**  
- Enter a **column number (1–3)**  
- The game updates the grid after every move
- It announces:
 - Player 1 wins  
 - Player 2 wins  
 - Draw  

# Requirements
- Python 3.6 or above  
- No external libraries required
